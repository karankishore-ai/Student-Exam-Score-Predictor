# Student Exam Score Predictor

## Project Overview
This project predicts a student's exam score based on study hours and attendance percentage using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Predicts exam scores
- Uses Linear Regression
- Simple and beginner-friendly

## Dataset
The dataset contains:
- Study Hours
- Attendance Percentage
- Exam Score

## How to Run
1. Install required libraries
2. Run: python exam_score_predictor.py
3. Enter study hours and attendance percentage
4. View predicted score
